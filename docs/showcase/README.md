# RF Drone Detection Showcase

This folder contains a standalone project showcase page for the
`experiment/stft-128-hop32` branch.

## Run Locally

From this folder:

```bash
python3 -m http.server 8015
```

Then open:

```text
http://127.0.0.1:8015
```

You can also open `index.html` directly in a browser.
