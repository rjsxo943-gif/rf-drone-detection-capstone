## 3. 문서에 반영한 핵심 메시지

현재 구현은 완성형 대드론 장비가 아니라, 예산과 장비 제약을 반영한 축소형 RF 탐지 프로토타입이다.

현재 단계의 목표는 다음을 검증하는 것이다.

```text
저비용 SDR과 소프트웨어 파이프라인만으로
2.4GHz RF 신호를 수집하고,
의미 있는 RF block을 선별하고,
spectrogram으로 변환하고,
CNN 분류 입력으로 연결할 수 있는가?
```

또한 현재 구조는 RF 수신부, 전처리, Energy Detection, STFT feature extraction, CNN classification, 방향 추정 branch로 분리되어 있으므로, 예산이 확보되면 다음 방향으로 확장 가능하다고 정리했다.

- 수평 안테나 배열 확장으로 360도 전방위 탐지
- 수직축 안테나 추가로 고각 판단
- 2.45GHz 근처 제한 대역에서 더 넓은 RF 대역 감시로 확장
- 다중 SDR 또는 동기화 수신기 기반 수신 구조 확장
- Drone-like / controller / telemetry 등 실제 드론 관련 클래스 추가
