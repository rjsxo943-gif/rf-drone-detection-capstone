## 비고

- 소프트웨어(GNU Radio, TensorFlow, Python)는 전량 무료 오픈소스
- 시연용 드론은 팀원 보유 또는 학과 대여 활용 (완구 드론 가능)
- 라즈베리파이4B 중고 구매 시 4GB 모델 확인 필수

## 시스템 패키지 설치 (Ubuntu 기준)

```bash
# GNU Radio
sudo apt install gnuradio

# PLUTO+ libiio 드라이버
sudo apt install libiio-dev
sudo apt install python3-libiio

# 라즈베리파이4B 추론 환경
pip install tflite-runtime
```
